# Multiple Disease Predictor
## About
This webapp was developed using Flask Web Framework. The models used to predict the diseases we trained on large Datasets. This webapp can predict following Diseases:
* Diabetes
* Breast Cancer
* Heart Disease
* Kidney Disease
* Liver Disease
* Malaria Disease
* Pneumonia Disease
<br>
